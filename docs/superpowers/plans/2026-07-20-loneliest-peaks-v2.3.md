# The Loneliest Peaks v2.3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Retire the "Edwards Polygon" name, fix the broken spotlight dim by rendering a precomputed complement polygon, make the hero and info panel collapsible, and soften two copy items.

**Architecture:** The spotlight fix moves the complement geometry from a fragile in-browser rectangle-with-hole to a pipeline-computed star-shaped polygon about each summit's antipode, reusing the existing ring machinery; the front-end just renders it. Everything else is copy/CSS/JS.

**Tech Stack:** Python 3.12 + NumPy + shapely (existing, offline) for the pipeline; vanilla ES modules + MapLibre GL for the front-end; pytest + Playwright.

**Spec:** `docs/superpowers/specs/2026-07-20-loneliest-peaks-v2.3-design.md` (read it first).

```python
def build_star_geometry(hub_lat, hub_lon, theta, R):
    """Valid GeoJSON geometry for a star-shaped region about (hub_lat, hub_lon)
    with boundary angular distance R(theta) in radians. Handles the single-pole
    and both-poles (world-with-hole) cases. Returns a GeoJSON geometry dict."""
    lats, lons = cell_ring(hub_lat, hub_lon, theta, R)
    north, south = poles_inside(hub_lat, np.mod(theta, 2 * np.pi), R)
    if north and south:
        span = float(np.max(lons) - np.min(lons))
        assert span < 350.0, "antipodal blob crossing the antimeridian is not supported"
        hole = list(zip(lons.tolist(), lats.tolist()))
        hole.append(hole[0])
        shell = [(-180.0, -90.0), (180.0, -90.0), (180.0, 90.0), (-180.0, 90.0), (-180.0, -90.0)]
        geometry = mapping(Polygon(shell, holes=[hole]))
    else:
        geometry = ring_to_geojson_geometry(lats, lons, north, south)
    from shapely.geometry import shape
    assert shape(geometry).is_valid, "star geometry must be a valid polygon"
    return geometry


def dominance_complement(hub_lat, hub_lon, theta, R):
    """GeoJSON geometry of the region OUTSIDE the dominance ring (everything
    beyond R(theta) from the hub). It is star-shaped about the hub's antipode,
    so it is built with the same robust machinery centred there."""
    lats, lons = cell_ring(hub_lat, hub_lon, theta, R)
    anti_lat = -hub_lat
    anti_lon = ((hub_lon + 360.0) % 360.0) - 180.0
    d_prime, phi = angular_distance_and_bearing(anti_lat, anti_lon, lats, lons)
    order = np.argsort(phi)
    return build_star_geometry(anti_lat, anti_lon, phi[order], d_prime[order])
```
